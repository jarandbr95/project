Git/github

Git er et distribuert versjonskontrollsystem, et verktøy for å administrere kildehistorikken din. Mens Github er en "hosting service" for Git repositories. Så Git er verktøyet, mens Github er tjeneren for prosjekter som bruker Git. Github gir deg muligheten for å samarbeide med andre i et prosjekt. Man kan gi andre tilgang til filene sine. Da har de muligheten for å se filene dine og gjøre endringer og laste dette opp på ny. Da vil du få en varsel om at den personen har gjort endring og kan gå inn og sjekke dette ut. Github gir deg også muligheten for å se på og gjenopprette tidligere versjoner. Github gir deg en oversikt over hele historien din og gir deg en backup av filene dine. Man kan sammenligne Github med Dropbox bare at Github er for kode.
